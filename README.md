## 🔧 Self-Host

```bash
git clone https://github.com/DasFletchi/NoLoginChat.git
cd NoLoginChat
npm install
npm run dev
```

### Add your own API key

Edit `src/App.jsx` and replace the API key:

```javascript
const API_KEY = 'your-key-here'
```

Get free key at: https://openrouter.ai
