1. Go to: https://github.com/DasFletchi/ScofieldMoney-test/settings/pages
2. Select: **Source → Deploy from a branch**
3. Branch: **main**
4. Folder: **/(root)**
5. Click **Save**
6. Wait 2-3 minutes for deployment

---

## What Was Built

- ✅ React + Vite Chat App (no login required)
- ✅ OpenRouter API (Llama 3.1) for AI responses
- ✅ Session-based (URL hash = session ID)
- ✅ localStorage persistence
- ✅ Mobile-friendly dark theme
- ✅ SEO keywords list
- ✅ 10 social post drafts
- ✅ Ad copy & placement plan
